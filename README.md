# Matrix Visual Effects

Small visual experiments inspired by the digital code aesthetics from *The Matrix (1999)*.

Both effects are written in **vanilla JavaScript using the HTML5 Canvas API** and run directly in the browser.

---

## Matrix Rain

A recreation of the famous **Matrix digital rain** effect.

Features:

- procedural falling character streams
- multiple streams per column
- randomized speed and stream length
- fading trail effect

Interaction:

When you **click on the screen**, a hidden word can appear and assemble itself inside the rain from the falling characters.

Open in browser:

matrix-rain.html

---

## Matrix Cipher

A numeric transformation animation where **digits rapidly mutate before resolving into readable numbers**.

This type of visual effect can be briefly seen **at the very beginning of *The Matrix (1999)* during Trinity's phone call**, where streams of numbers appear on the screen before switching to the Matrix code.

Open in browser:

matrix-cipher.html

---

## How to run

Simply open the HTML files in your browser.

No installation or dependencies required.
